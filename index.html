from gtmdb import connect_gtmdb

db, scope = await connect_gtmdb(api_key=api_key)
await db.leads.create(
    scope, actor_id="my-agent",
    company_name="Acme", status="new",
)
await db.close()

Built for autonomous GTM

  • Commercial graph

    Nodes and relationships in Neo4j—360° views, explore subgraphs, and attribution-style paths on the same store sales and marketing share.

  • Scope & policies

    Tenant isolation, read/write rules, field-level masking, and redaction modes—different tokens for reps, internal agents, or partners.

  • Audit trail

    Activity logging for API requests (with reasoning where provided) so you can trace who did what, including scoped self-serve log queries.

  • Search & explore

    Full-text search and graph exploration endpoints so agents can navigate the pipeline without ad-hoc SQL across silos.
