# GTMDB documentation

- **[api-reference.html](api-reference.html)** — static API reference (open in a browser). The left sidebar uses **[assets/gtm-db-logo.png](assets/gtm-db-logo.png)** (**gtmDB**, transparent PNG); keep the `assets/` folder next to the HTML file when sharing. Includes a GtmDB product intro, **permissions / scopes / tokens** (policies, presets, masking, redaction), then `GtmDB` methods, entity fields, parameters, and Python examples. Jaeger-inspired dark theme.

To preview locally:

```bash
open docs/api-reference.html   # macOS
xdg-open docs/api-reference.html  # Linux
```
